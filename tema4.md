# Огляд та характеристики слотів розширення (expantion slots) на материнській платі


## PCI

(Peripheral Component Interconnect) — це стандартна шина вводу/виводу, розроблена Intel, для підключення периферійних пристроїв (відеокарт, звукових мережевих карт) до материнської плати комп'ютера.

### Специфікація шини PCI -

   - частота шини — 33,33 МГц або 66,66 МГц, передача синхронна;
   - розрядність шини — 32 або 64 біт, шина мультиплексована (адреси і -дані передаються по спільній шині);
   - пікова пропускна здатність для 32-розрядного варіанту, що працює на частоті 33,33 МГц — 133 Мб за секунду;
   - адресний простір пам'яті — 32 біт (4 байти);
   - адресний простір портів вводу-виводу — 32 біт (4 байти);
   - конфігураційний адресний простір (для однієї функції) 256 байтів;
   - напруга 3,3 або 5 вольт. 

### Стандартні модификації PCI - 

- PCI 2.0 — перша версія базового стандарту, що одержала широке поширення, використовувалися як карти, так і слоти із сигнальною напругою тільки 5V. Пікова пропускна здатність — 133 Мб/с;
- PCI 2.1-3.0 — відрізнялися від 2.0 можливістю одночасної роботи декількох пристроїв bus-master (так званий конкурентний режим), а також появою універсальних карт розширення, здатних працювати як на 5V, так і на 3.3V слотах (з частотою 33 і 66 МГц відповідно). Пікова пропускна здатність для 33 МГц — 133 МБ/с, а для 66 МГц — 266 МБ/с;

## PCI-X

Peripheral Component Interconnect eXtended - це стандарт комп'ютерної шини і плати розширення, який покращує 32-бітну локальну шину PCI для більш високої пропускної спроможності, необхідної в основному для серверів та робітників станцій. Він використовує модифікований протокол підтримки більш високих тактових частот (до 133 МГц), але у іншому схожий електричну реалізацію. PCI-X 2.0 збільшив швидкість шини до 533 МГц зі зниженням рівня електричного сигналу.

Слот фізично являє собою слот PCI на 3,3 В з таким самим розміром, розташуванням і призначенням контактів. Електричні специфікації сумісні, але вони мають суворіші вимоги. Однак, у той час як більшість звичайних слотів PCI є 32-розрядною версією довжиною 85 мм, більшість пристроїв PCI-X використовують 64-розрядний слот довжиною 130 мм, і 64-розрядні роз'єми PCI і підтримка PCI-X стали розглядаються як синоніми.

PCI-X фактично повністю визначений як для 32-, так і для 64-бітових роз'ємів PCI, а PCI-X 2.0 додав 16-бітний варіант для додатків, що вбудовуються.

## PCI-e 

комп'ютерна шина, що використовує програмну модель шини PCI і високопродуктивний фізичний протокол, заснований на послідовній передачі даних.
Започаткована у липні 2002 року компанією Intel. 

На відміну від шини PCI, що використала для передачі даних спільну шину, PCI Express, в загальному випадку, є пакетною мережею з топологією типу зірка, пристрої PCI Express взаємодіють між собою через середовище, утворене комутаторами, при цьому кожен пристрій безпосередньо зв'язаний з'єднанням типу точка-точка з комутатором.

Крім того, шиною PCI Express підтримується:

   - гаряча заміна карт;
   - гарантована смуга пропускання (QoS);
   - управління енергоспоживанням;
   - контроль цілісності передаваних даних.

Розробку стандарту PCI Express почала фірма AMD після відмови від шини InfiniBand. Офіційно перша базова специфікація PCI Express з'явилася в липні 2002 року.

 ![Photo](https://globalamericaninc.com/wp-content/uploads/2021/07/PCI_und_PCIe_Slots.jpeg)

- **Лінії (lanes)**  
  PCI-e складається з одного або кількох каналів передачі даних:
  - **x1** — одна лінія
  - **x4** — чотири лінії
  - **x8** — вісім ліній
  - **x16** — шістнадцять ліній  
  Кожна «лінія» — окремі пари провідників для прийому та передачі даних.

- **Версії стандарту**  
  Кожне наступне покоління підвищує пропускну здатність однієї лінії:
  - PCI-e 1.0 — близько 250 МБ/с на лінію в одному напрямку
  - PCI-e 2.0 — 500 МБ/с
  - PCI-e 3.0 — ~1 ГБ/с
  - PCI-e 4.0 — ~2 ГБ/с
  - PCI-e 5.0 — ~4 ГБ/с  

## AGP

AGP (Accelerated Graphics Port) — застарілий інтерфейс для підключення відеокарт до материнської плати. Використовувався переважно з кінця 1990-х до середини 2000-х років.

Призначений тільки для відеокарт. Мав окремий канал передачі даних, що давало вищу продуктивність порівняно з PCI.

Режими роботи  
AGP 1x — 266 МБ/с  
AGP 2x — 533 МБ/с  
AGP 4x — 1,07 ГБ/с  
AGP 8x — 2,1 ГБ/с  

Підтримував різну напругу живлення 3.3 В 1.5 В 0.8 В тому роз’єми мали ключі для сумісності.

Згодом був замінений PCI-Express який забезпечив більшу швидкість та універсальність.

# Карти розширення (expantion cards) на інтерфейсі PCI-e 

PCI-e (Peripheral Component Interconnect Express) — сучасний інтерфейс для підключення плат розширення до материнської плати. Забезпечує високу швидкість передачі даних і підтримує різну кількість ліній x1 x4 x8 x16.

## Основні типи карт

Відеокарти  
Найчастіше використовують слот x16. Забезпечують обробку графіки та 3D-рендеринг.

Мережеві карти  
Ethernet або Wi-Fi адаптери зазвичай формату x1 або x4.

Звукові карти  
Покращують якість аудіо порівняно з інтегрованими рішеннями.

SSD адаптери  
Плати для підключення NVMe накопичувачів через PCI-e.

RAID контролери  
Дозволяють підключати кілька дисків і створювати RAID масиви.

Плати розширення портів  
USB SATA Thunderbolt та інші інтерфейси.

## Переваги PCI-e

Висока пропускна здатність  
Масштабованість за кількістю ліній  
Зворотна сумісність між версіями  
Низькі затримки передачі даних

PCI-e є основним стандартом для сучасних комп’ютерних систем.

## PCI-e x1
Найкоротший слот. Використовується для пристроїв з невеликою пропускною здатністю: мережеві карти Ethernet, Wi-Fi адаптери, звукові карти, плати розширення USB.

 ![Photo](https://content2.rozetka.com.ua/goods/images/big_tile/446653873.jpg)


## PCI-e x4
Середній за довжиною слот. Використовується для SSD адаптерів NVMe, RAID контролерів початкового рівня, мережевих карт 10GbE.

## PCI-e x8
Застосовується для високошвидкісних мережевих карт 10–25GbE, професійних контролерів зберігання, деяких серверних плат розширення.


## PCI-e x16
Найдовший і найпоширеніший слот. Переважно використовується для відеокарт. Також підходить для інших високопродуктивних пристроїв.

 ![Photo](https://riotpixels.net/wp/wp-content/uploads/2015/03/pcie_slots400.gif)


## Mini PCI-e
Компактна версія для ноутбуків і вбудованих систем. Використовується для Wi-Fi, Bluetooth, модемів.

 ![Photo](https://www.gsmsota.ua/wa-data/public/shop/products/00/webp/15/11/1115/images/4342/4342.970.webp)

## M.2 (через PCI-e)
Сучасний компактний інтерфейс, який використовує лінії PCI-e. Найчастіше застосовується для NVMe SSD, інколи для бездротових модулів.
