# Характеристики оперативної пам'яті типів DDR4 та DDR5, їх порівняння
### DDR4 SDRAM
**Синхронна динамічна оперативна пам'ять DDR4**

- Максимальна місткість накопичувача DDR4 SDRAM в чотири рази більша
- Споживає менше енергії, ніж DDR3 (1,2 В)
- Працює на більш високих тактових частотах (до 1600 МГц)
- Роз’єм має 288 контактів
- Доступна з розширеними функціями виправлення помилок, такими як помилкова кодова пам'ять (пам'ять ECC) для виявлення декількох бітових помилок

---

# DDR5 SDRAM
**Синхронна динамічна оперативна пам'ять DDR5**

- Вища пропускна здатність порівняно з DDR4
- Працює на ще вищих тактових частотах (від ~4800 МГц і вище)
- Менше енергоспоживання завдяки нижчій напрузі (близько 1,1 В)
- Вбудований контролер живлення на модулі (PMIC)
- Підтримка покращеної корекції помилок (on-die ECC)
- Вища щільність пам’яті (більший об’єм модулів)
- Новий роз’єм із 288 контактами, але фізично несумісний з DDR4
- Оптимізована для сучасних процесорів та високонавантажених систем

# Порівняння DDR4 та DDR5

## Технічні характеристики

| Характеристика | DDR4 | DDR5 |
|------|------|------|
| Стартова частота | ~2133–3200 MHz | від ~4800 MHz |
| Напруга | ~1.2 В | ~1.1 В |
| Пропускна здатність | Середня | Висока |
| Енергоспоживання | Вище | Нижче |
| Контролер живлення | На материнській платі | На модулі (PMIC) |
| Об’єм модулів | Менший | Більший потенціал |
| Сумісність | Старіші платформи | Лише нові платформи |

---

## Практичне порівняння

### DDR4
- Дешевший
- Широка сумісність з платами
- Достатній для ігор, навчання, офісних і робочих задач
- Оптимальний для бюджетних і середніх ПК

### DDR5
- Дорожчий
- Вища швидкість передачі даних
- Краще підходить для нових процесорів
- Перспективний стандарт для майбутніх апгрейдів

---

# Висновки

- **DDR4** — раціональний вибір для недорогих та середніх збірок, де важлива ціна і сумісність.
- **DDR5** — вибір для сучасних платформ, високопродуктивних систем і майбутньої модернізації.
